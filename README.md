# Employee Vacation Tracker

A simple web app to track employee vacation requests, annual allowance, and remaining days.

## Features
- Add employee vacation requests with start/end dates.
- Automatically compute remaining vacation days.
- Status badges for healthy/watch/low remaining balance.
- Delete individual requests or clear all.
- Saves data in browser `localStorage`.

## Run locally
Open `index.html` directly in a browser, or run a static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
